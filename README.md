Project php_learn 
Cloud Storage

Облачное хранилище для файлов различного типа. Максимальный размер файла 2 Гб.
Реализовано:
- шаринг папок и файлов как для пользователей, так и для групп;
- смена  пароля пользователем;
- редактирование пользователей администратором;
- редактирование групп администратором;
- сброс пароля (default = 'user123') или задание нового.


init.sql - развертывание бд для работы 
config/database.php - настройки для подключения к бд 
public/index.php - точка входа 

для входа: 
login - admin
passwd - 123
